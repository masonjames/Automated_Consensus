# Next.js Frontend for step01_synthesize_personas.py

This project is a frontend application built with Next.js to interact with the Python script `step01_synthesize_personas.py`.

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
